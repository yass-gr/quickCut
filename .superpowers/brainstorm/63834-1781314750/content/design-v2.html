Font Strategy

JetBrains Mono for brand consistency + a display font for scroll-stopping hooks

A

JetBrains Mono only — pure brand consistency

Every text element in the video uses JetBrains Mono. Distinctive, techy, 100% aligned with MSSOUGRA. Less visually varied but maximum brand reinforcement.

B

JetBrains Mono for data + Inter for hooks

Hooks in Inter Bold (clean, readable, modern sans-serif). Stats/predictions/labels in JetBrains Mono. Gives hooks more visual punch while keeping data feeling analytical.

C

Bespoke: JetBrains Mono for stats + display font for hooks

Hooks in a sporty condensed display font (e.g., Bebas Neue, Anton, or Oswald). Maximum energy. Data in JetBrains Mono for credibility. Best for engagement but most font files.
