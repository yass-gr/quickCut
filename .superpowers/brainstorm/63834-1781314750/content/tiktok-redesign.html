The New Approach

Background Image

A single full-frame background image (player photo, match action, stadium) that you select. Same image across all 5 scenes for consistency. Dark gradient overlay ensures text readability.

Text Over Image

Text sits on top of the image, not inside cards. Semi-transparent backdrop/blur behind text blocks for readability. Green accent color for key numbers and labels.

No Card UI

No #101010 card backgrounds, no border boxes. Stats use thin semi-transparent pills. Everything floats over the image. Feels like video content, not an app.

Darija Hooks Big

Hook text takes up ~60% of the screen. Big, bold, scroll-stopping. Bottom third reserved for stats/prediction. Watermark persistent top-right in small text.

Background Image Option

You pick one. It sits behind all 5 scenes. Overlay varies per scene.

A

Single image — same across all scenes

Pick one player photo or match photo. Dark gradient overlay adjusts per scene (heavier on text-heavy scenes, lighter on logo scenes). Simple to implement, consistent look.

B

Per-scene images

Different image per scene (hook has one photo, match intro has logos only, etc.). More dynamic but more images to source.

C

Background video loop

Short video clip playing behind the text. Most engaging but heaviest on rendering and file size.
