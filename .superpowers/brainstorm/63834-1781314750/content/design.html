Video Canvas Options

The dark canvas sets the mood. Let's dial in the colors.

A

Deep Navy + Accent

Dark, premium feel. Red accent pops against deep blue. Same vibe as Google Pixel launch videos.

B

Pure Black + Neon

True black (OLED-friendly). Neon cyan and pink accents. More energetic, sporty feel.

C

Dynamic — Team Colors

Neutral dark base, but gradient/highlight colors pulled from home team logo. Each video looks unique.

Font Options

Spec says Montserrat. But we have choices within that family + pairings.

A

Montserrat (all weights)

Clean, geometric, versatile. Bold 700 for hooks, Medium 500 for stats, Regular 400 for body. Matches the modern sporty feel.

B

Inter + Montserrat

Inter for UI (wizard), Montserrat for video. Inter is more legible at small sizes — better for the wizard. Montserrat stays for the dramatic video text.

C

Plus Jakarta Sans

Similar to Montserrat but with more personality. Slightly condensed, very modern. Works for both UI and video. One font for everything.

Wizard UI Theme

Separate from the video canvas — this is what the user interacts with.

A

Dark UI — matches video vibe

Dark mode app that previews the dark video in context. Immersive. No context-switching.

B

Light UI — clean productivity

Apple-style clean white interface. Easier to read form fields. Video preview is full-contrast against white.

C

System auto (follows OS theme)

Both themes, responds to `prefers-color-scheme`. Best for developer comfort. More work to maintain.
